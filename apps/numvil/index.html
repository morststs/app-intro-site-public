Numvil

ブラウザ

数字と記号のパーツを「式」に並べて「題」の数を作る計算パズル

名前は num（数）＋ anvil（金床）＝「数を鍛えて組み立てる」より。

どんなゲーム？

  • 「題」に表示された数を、数字や記号の「パーツ」を並べた式で作る計算パズル。
  • 画面は「題」「パーツ」「式」「結果」の4つのエリアに分かれています。
  • 式を組み立てると計算結果がその場で表示され、題の数と一致するとクリア。

操作

  • パーツ（0-9 と ( ) ! + - / * ^）をドラッグ／タップで「式」に並べる。
  • 式の中に置いたパーツはドラッグで並べ替えできる。
  • 「Clear all」で式をリセット、「New」で次の問題、「Give up」で解答例を表示。

ルール

  • パーツは各1個ずつ。1つの式の中で同じパーツは1回しか使えない。
  • 数字を2つ並べて2桁以上の数（例：1 と 0 で 10）を作ることはできない。
  • 計算の順序はふつうの数学と同じ（階乗 ! → べき乗 ^ → かけ算・わり算 → たし算・ひき算）。
  • どの問題にも、必ず正解となる式が1つ以上あります。

モードと難易度

  • Endless（エンドレス）／ Challenge（チャレンジ）のモードを切り替え。
  • Endlessモードの難易度は Easy / Normal / Expert の3段階。
  • Challengeモードは レベル１～５まであり、各レベルは 10 問で構成。
